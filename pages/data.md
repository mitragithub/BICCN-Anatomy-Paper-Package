# Available Data Sets

## Mouse Head Imaging Data
Comprehensive collection of standardized datasets for the Brain Architecture Project.

### Categories
- Raw Imaging Data
  - MRI scans
  - CT scans
  - Histology images
  
- Processed Data
  - Segmentation maps
  - Registration results
  - Feature maps

### Access
All datasets are available under GNU GPL v3 license.
